Desktop media conversion, without the cloud

Convert images and videos locally with a queue that stays fast, clear, and offline.

Flux Converter packages FFmpeg into a native desktop workflow for Windows, macOS, and Linux. Drop files in, choose output formats, run conversions in parallel, and keep everything on your machine.

Workflow

Built for a short path from import to export.

Flux keeps the workflow direct: select files, confirm output, queue conversions, and reveal finished media without bouncing through separate tools.

01
Add files

Use drag and drop or the native file picker.

02
Probe and validate

03
Queue in parallel

Set concurrency from one to four simultaneous jobs.

04
Reveal output

Supported formats

Common formats in, practical formats out.

Output

Acceleration

NVIDIA NVENC, AMD AMF, Intel QSV, Apple VideoToolbox, and Windows Media Foundation when available.
